# Tutorial: Understanding, Integrating, and Discussing Data with LLMs

This tutorial explores data understanding and some of its applications, leveraging semantic annoations. Semantic annoations are inferred by mining Wikidata using [TorchicTab-Heuristic](https://github.com/dtai-kg/TorchicTab-Heuristic). 
Then the tutorial explores:
- Data enrichment leveraging the inferred annoations
- Automated knowledge graph construction with [AutoRML](https://github.com/dtai-kg/AutoRML)

## Installation: 

    pip install autorml
